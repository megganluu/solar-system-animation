## Description
> This project is an interactive solar system physics-based animation project. Planets mimic orbiting around the sun via trigonometric functions.
> Planets vary in color, shape, orbit, and shading around a light emitting sun.
### Keyboard Controls
- 0: Bird's eye view
- 1: Planet 1 - Flat Shaded Blocky 
- 2: Planet 2 - Dynamic Shaded Smooth 
- 3: Planet 3 - Saturn (wobbling rings)
- 4: Planet 4 - Smooth Blue
- 5: Planet 4's Moon - Diamond Moon orbiting P4
  
## Computer Graphics Topics
- UV mapping
- Rigging
- Shading
- Camera angle
- User Interaction
## Tech Stack
- Vite
- Three.js
## How to run on local
```
git clone https://github.com/megganluu/solar-system-animation
cd solar-system-animation
npm install
npm run dev
```
